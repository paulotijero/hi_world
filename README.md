# hi_world

This is my first published gem and says "Hi world!"

## Installation

The first step is adding the gem in your Gemafile

```
gem "hi_world"
```

Use the package manager `bundle` to install hi_world.

```bash
bundle install
```

## Usage

```ruby
require 'hi_world'

puts Hi_world.hi
```
